// Пометка, что класс сущность
@Entity

// Данные о таблице
@Table(name="messages")

// Первичный ключ, обеспечить уникальность и автоинкремент на стороне базы
@Id

// Внешний ключ
@ForKey

// Метод генерации первичного ключа
@GeneratedValue

// Данные о колонке, добавить возможность делать уникальной данные столбца, если name пустой, то автоматом мапить в БД название столбца как у поля.
@Column(name="username", unique=true)

// Значение не может быть пустым
@NotNull

//Виды связи, учесть каскад
@OneToOne (связь один к одному, то есть один объект Entity может связан не больше чем с один объектом другого Entity );
@OneToMany (связь один ко многим, один объект Entity может быть связан с целой коллекцией других Entity);
@OneToMany(mappedBy = "user", cascade = CascadeType.ALL, orphanRemoval = true)
@ManyToOne (связь многие к одному, обратная связь для OneToMany);
@ManyToMany (связь многие ко многим).

//Нужно для определения связи
@JoinColumn — аннотация используется для указания столбца FOREIGN KEY, используемого при присоединении к ассоциации объекта или встраиваемой коллекции.

Под вопросом, взято из JPA:
@Basic — аннотация используется для сопоставления базового типа атрибута столбцу таблицы базы данных.
@ColumnResult — аннотация @ColumnResult используется в сочетании с аннотациями @SqlResultSetMapping или @ConstructorResult для отображения столбца SQL для заданного запроса SELECT.

@JoinColumns — аннотация используется для группирования нескольких аннотаций @JoinColumn, которые используются при сопоставлении объектов или
 встраиваемой коллекции с использованием составного идентификатора.

@MapKey — аннотация используется для указания ключа ассоциации java.util.Map, для которой ключ является либо первичным ключом, либо атрибутом объекта, который представляет значение Map.
@Transient — аннотация используется для указания того, что данный атрибут сущности не должен сохраняться.
@UniqueConstraint — аннотация используется для указания уникального ограничения, которое должно быть включено генератором автоматической схемы для первичной
или вторичной таблицы, связанной с текущим аннотированным объектом.
@Generated — аннотация используется, чтобы указать, что текущий аннотированный атрибут объекта генерируется базой данных.
